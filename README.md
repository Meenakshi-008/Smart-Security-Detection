# Smart Security Detection System

**Person detection and counting using deep learning**

Pre-trained Faster R-CNN (PyTorch) detects people in uploaded images, draws confidence-aware bounding boxes, counts individuals, and displays results via a clean Gradio web interface.

### Key Features
- Accurate person detection  
- Bounding boxes + confidence scores  
- Automatic person counting  
- Simple image upload & result display  

### Tools & Technologies
- Python  
- PyTorch + torchvision (Faster R-CNN)  
- OpenCV  
- Gradio
  
### Installation and Setup

Follow these steps to set up and run **SmartSecurityGuard** locally:

1. Clone the repository -  git clone https://github.com/Meenakshi-008/SmartSecurityGuard.git cd SmartSecurityGuard
2. Create and activate a virtual environment.
3. pip install --upgrade pip
   pip install -r requirements.txt
4. python app.py
